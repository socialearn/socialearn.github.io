12. QUIZZ05 - 02 - 2023|2x + 5| = 7berapakah himpunan penyelesaian?​


Materi : Konsep Mutlak

|2x + 5| = 7

(1) f(x) = 7

2x + 5 = 7

2x = 7 - 5

2x = 2

x = 1

(2) - f(x) = 7

-( 2x + 5 ) = 7

- 2x - 5 = 7

2x = - 7 - 5

2x = -12

x = -6

HP = { x | x = -6 atau x = 1 }

[tex]\sf \bf \sf (7! - 4!) + 5![/tex]

[tex]\sf \bf \sf ((840×6×1) - 24) + (60×2)[/tex]
