3. mr. galang has been studying medical technology for doctorate program at columbia university for about two years. we can also say : medical technology ... by mr. galang for doctorate program at the university.


Jawaban:

The question seems to be asking about passive judging from the gap so it should be 'Medical technology has been being studied by Mr.Galang for doctorate program at the university.'

Pembahasan:

The tense in the original sentence is present perfect continous. It can be seen from the pattern of it

Mr Galang + has been + studying + medical technology for doctorate program at Columbia University for about two years.

S + has/have been + Ving + O + complement

Such tense is used to tell about an activity that starts to be done in the past but still ongoing up to now. This tense usually focus on the duration of such activity, not the result.

To transform it to passive voice, the order in the pattern has to be changed into like this below because in passive sentence, the object comes first and more important than the subject.

O + has/ have been + being + V3 + by S

So, it will be 'Medical technology has been being studied by Mr.Galang for doctorate program at the university.'
