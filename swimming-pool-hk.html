29. ( + ) Adi is swimming in swimming pool( - ) ................................................ ( ? ) ................................................?​


Jawaban:

( - ) Adi is not swimming in swimming pool

( ? ) Is Ali swimming in swimming pool?

Topik :

Present Continuous Tenses

Rumus :

VERBAL

(+) S + Is/Am/Are + Ving + O

(-) S + Is/Am/Are + Not + Ving + O

(?) Is/Am/Are + S + Not + Ving + O

NOMINAL

(+) S + Is / Am / Are + 3C

(-) S + Is/Am/Are + 3C

(?) Is/Am/Are + S + 3C

Time Signal :

- Now, Right Now, still , at the moment, listen!, look!

Penjelasan :

Nominal Present Continuous sama dengan Nominal Simple Present

Catatan :

3C ( Adjective,Noun,Adverb )

Jawaban:

( + ) Adi is swimming in swimming pool

( - ) Adi is not swimming in swimming pool

( ? ) Is Adi swimming in swimming pool?
