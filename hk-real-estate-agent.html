Ubahlah kalimat berikut kedalam bentuk pasif yang tepat untuk setiap kalimat present continuous tense di bawah ini 3.We are not currently accepting new users. ................................................................................................................................................... 4..The real estate agent is selling our properties. ...................................................................................................................................................


Jawaban:

3. New users are not currently being accepted by us.

4. Our properties are being sold by the real estate agent.

Penjelasan:

Kalimat aktif: (Simple Present Continuous)

Subject + to be + V-ing + Object

Kalimat pasif: (Simple Present Continuous)

Subject + to be + being + V-3 (past participle) + by + Object

Kuncinya:

Subjek dalam kalimat aktif dijadikan Objek dalam kalimat pasif.

Objek dalam kalimat aktif dijadikan Subjek dalam kalimat pasif.

Langkah pertama dalam mengubah kalimat aktif menjadi pasif atau sebaliknya adalah harus lebih dulu menemukan Subjek dan Objeknya.

Langkah selanjutnya adalah mengikuti rumus di atas.

Dan, satu lagi yang penting, harus hafal bentuk-bentuk kata kerja Inggris, bentuk present (I), past (II), dan past participle (III). Ini tidak boleh ditawar. Harus hafal.

Rumus juga harus dihafalkan.

6. The number of houses sold by an estate agent follows a Poisson distribution with a mean of 2 per week. Find the probability that in the next 4 weeks, the estate agent sells more than 6 houses. ​


Jawab:

An average of 2 per week means that the agent on average sells 8 houses per month (per 4 weeks). Therefore, we are going to compare the probability of the agent selling more than 6 houses per month given the average of 8 per month (λ = 8).

P(x = k) = λ^k × e^(-λ) ÷ k!

P(x = 0) = 8^0 × e^(-8) ÷ 0!

P(x = 1)  = 8^1 × e^(-8) ÷ 1!

P(x > 6) = 1 - P(x ≤ 6)

P(x > 6) = 1 - [P(x = 0) + P(x = 1) + P(x = 2) + P(x = 3) + P(x = 4) + P(x = 5) + P(x = 6)]

P(x > 6) = 1 -  P(x <= 6)

P(x > 6) = 1 - 0.31337

P(x > 6) = 0.68663

Jawaban:

Modals

If she wants to buy an apartment, she........... a good real estate agent.

a. ought to not consult

b. ought not to consult

c. should consult

d. should not consult

=> If she wants to buy an apartment, she should consult a good real estate agent.

Penjelasan:

=>> Modal Auxiliary Verb merupakan kata kerja bantu dalam bahasa Inggris yang terdiri dari beberapa kata yaitu :

✓ Can (bisa/dapat) biasanya digunakan untuk :

>> menunjukan kemampuan

>> memberi pilihan

>> meminta izin

✓ May (mungkin) biasanya digunakan untuk :

>> menunjukan kemungkinan

>> meminta izin secara lebih sopan

✓ Must (harus/pasti) biasanya digunakan untuk :

>> keyakinan atas suatu hal

>> kemungkinan yang pasti terjadi

✓ Should (seharusnya/sebaiknya) biasanya digunakan untuk :

>> memberi saran (yang harus dilakukan/tidak dilakukan)

✓ Would (akan) biasanya digunakan untuk :

>> meminta izin atau menawarkan sesuatu

✓ Will (akan) biasanya digunakan untuk :

>> mengungkapkan rencana

✓ Shall (akan) biasanya digunakan untuk :

>> mengungkapkam rencana, sesuatu yang akan terjadi.

✓ Have to (harus/kewajiban) biasanya digunakan untuk :

>> mengingatkan kewajiban/janji seseorang

✓ Might (mungkin/boleh) biasanya digunakan untuk:

>> menunjukan kemungkinan yang lebih sedikit dari penggunaan auxiliary 'may'

✓ Ought to (sebaiknya/seharusnya) biasanya digunakan untuk :

>> Saran untuk melakukan sesuatu

✓ Could (bisa/dapat) biasanya digunakan untuk:

>> meminta izin lebih sopan

>> menunjukan kemampuan bentuk lapau

=> Sama seperti Tenses, Modal Auxiliary Verb juga memiliki rumus untuk affirmative, negative, dan interrogative nya. Rumusnya sbb:

(+)

[S + Modal + V1 + O/C]

(-)

[S + Modal + Not + V1 + O/C]

(?)

[Modal + S + V1 + O/C]

=> Perintah pada soal diatas :

[melengkapi kalimat dengan opsi a,b,c,d]

Jawaban yang tepat yaitu opsi C. Should consult.

=> Kalimat yang lengkap menjadi :

[If she wants to buy an apartment, she should consult a good real estate agent.]

terjemahannya :

{Jik dia ingin membeli sebuah apartemen, dia harus berkonsultasi dengan agen perumahan yang bagus}
