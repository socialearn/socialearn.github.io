22. bagaimana langkah langkah membuka program Corel draw X3 dari meu start​


Membuka program CoreldDRAW X3 pada komputer cukup mudah, berikut langkah-langkahnya:

Klik icon start yang terdapat pada bagian kiri bawah deskop. Kemudian akan muncul daftar program pada komputer.Carilah icon CoreldDRAW X3, kemudian klik 2 kali pada icon tersebut.Setelah itu, akan muncul logo CoreldDRAW X3 berbentuk kotak, tunggulah beberapa saat hingga program benar-benar terbuka.Saat program sudat terbuka, akan muncul kotak dialog "welcome to CorelDRAW X3. Untuk membuka lembar kerja baru, klik "new".Lembar kerja CorelDRAW siap untuk digunakan.

Pembahasan

CorelDraw X3 merupakan salah satu versi CorelDRAW. Program tersebut bergerak dalam editor grafik vektor. Perusahaan yang mengembangkan program ini bernama Corel. Perusahaan tersebut merupakan perusahaan perangkat lunak yang terletak di Ottawa, Kanada. Ada berbagai macam versi program Corel, versi terbarunya ialah CorelDRAW X8 yang dirilis pada tanggal 15 Maret 2016.

Program ini bisa kita akses melalui komputer. Langkah-langkah membukanya cukup mudah seperti yang sudah dijelaskan di atas. Program CorelDRAW X3 merupakan program untuk membuat gambar berbasis vektor, contohnya ilustrasi, kartun, logo dan lain sebagainya. Gambar yang dihasilkan program ini sangat baik, karenanya banyak orang yang menggunakan program ini.

28. langkah langkah membuka corel draw ?


click start , buka corel draw
Pertama tekan CTRL+N untuk membuat kanvas kerja baruPilih Rectangle Tool atau tekan F6.
Sambil menekan Ctrl, tekan dan geser Rectangle Tool untuk membuat kotak segiempat sama sisi (kubus).
Pastikan kubus dalam keadaan terpilih. Pada Property Bar, rubah ukuran lebar menjadi 5 CM dan tinggi 5 CM. Setelah selesai tekan Puntuk menempatkan kubus tepat di tengah kanvas kerja.
Tekan F4 untuk memperbesar tampilan kubus.
Pilih Shape Tool atau tekan F10. Kemudian tarik node yang ada di sudut   kiri atas ke arah dalam untuk membuat sudut-sudut kubus melengkung.    Tekan F11 untuk memunculkan kotak dialog Fountain Fill. Kemudian atur beberapa spesifikasi seperti contoh di bawah ini. Setelah selesai tekan OK.Pilih Pick Tool kemudian klik di sembarang tempat untuk melepaskan pilihan pada objek kubus.Pilih kembali Rectangle Tool atau tekan F6. Buat sebuah kotak persegi panjang dan tempatkan di posisi atas kubus.

Sama seperti pada objek kubus, lengkungkan sudut-sudut kotak persegi panjang tersebut menggunakan Shape Tool.
Tekan F11. Pada kotak dialog Fountain Fill atur beberapa spesifikasi seperti contoh di bawah ini. Setelah selesai tekan OK.Tekan Outline kemudian pilih None untuk menghilangkan garisoutline.
Pilih Transparency
Pada Property Bar, pilih type: Linear dan angle: -90.
Pilih Text Tool atau tekan F8. Tulis huruf "f" menggunakan font:Arial Bold, ukuran: 150 pt dan warna: putih. Geser dan tempatkan huruf dengan "f".
