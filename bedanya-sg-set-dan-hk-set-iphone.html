6. apa arti rubber set dan straight set?


Dalam permainan bulu tangkis ada banyak sekali macam kata yang dipergunakan, misansalnya seperti : rubber set serta straight set. Selain itu masih ada berbagai istilah lainnya.

Straight Set : Pertandingan yang terselesaikan pada 2 set pribadi. Rubber Set : Keadaan waktu dua set sebelumnya seimbang (saling mengalahkan), maka dilakukan set ke 3 menjadi penentuan set.

Penjelasan:ISTILAH SERVIS Server dan Receiver: dua istilah ini digunakan buat menyebut atlet yg melakukan serivis (server) serta penerima servis (receiver). Long, Short, serta Wide: dalam pertandingan buulutangkis, kita lebih mengenal dengan istilah long, short, serta wide untuk menyatakan servis yg gagal. Shuttlecock yang jatuh di depan service box lawan dinamakan short. umumnya wasit akan berkata “Service too short.” jika shuttlecock jatuh memanjang ke luar lapangan, maka dinamakan long, sementara bila melebar ke luar dinamakan wide. Service Over: digunakan ketika server kalah dalam sebuah rally. sehingga giliran servis harus berpindah ke versus. Let: Let diberikan ketika penerima servis belum siap sementara servis sudah dilakukan. jika dinyatakan ‘let’ pemain tidak menambah score-nya serta servis harus diulang.

ISTILAH SCORE Love: pada pertandingan bulutangkis, score 0 dinyatakan menjadi ’love’. All: Jika ke 2 (pasang) pemain mempunyai score yg sama maka dinyatakan menjadi ‘all’. contohnya waktu kedudukan score dua-2, maka dinamakan ‘two all’. Deuce: Bila kedudukan score 20-20 maka permainan dinyatakan menjadi ‘deuce’. merupakan, galat satu (pasang) pemain harus bisa memimpin dua poin lebih buat memenangkan pertandingan.

ISTILAH PERMAINAN Rally: sebuah rally pertandingan dimulai saat keliru satu pemain melakukan servis. Rally berakhir Bila bola menyentuh tanah atau Bila pemain dinyatakan fault. Pemenang rally akan diberi satu score. Game/ Set: pada pertandingan bulutangkis, kita harus mendapatkan score 21 buat memenangkan pertandingan. Bulutangkis mempunyai 2-tiga sesi pertandingan, satu sesi pertandingannya dinamakan sebagai set atau game. Match: pertandingan bulutangkis memiliki 2-tiga game/set. Pemain dinyatakan memenangkan pertandingan Jika dia berhasil menang 2 set. tapi Jika ke 2 pemain masing-masing memenangkan satu game, maka wajib diadakan game lanjutan yg dinamakan match menjadi penentu pemain mana yg akan keluar menjadi pemenang. Straight Set : Pertandingan yang terselesaikan pada 2 set pribadi. Rubber Set : Keadaan waktu dua set sebelumnya seimbang (saling mengalahkan), maka dilakukan set ke 3 menjadi penentuan set. Stroke/ pukulan: saat pemain mengayunkan raket, maka dianggap menjadi ‘pukulan’ atau ‘stroke’. Clear Lob: memukul shuttlecock setinggi ke arah versus dinamakan menjadi ‘lob'. Drop Shot: pukulan yg enggak terlalu keras serta menunjuk ke lapangan lawan. Smash: pukulan yg sifatnya kencang dan ofensif ke arah lawan. umumnya arah pukulan smash menukik dengan cepat. Drive: pukulan cepat tapi mendatar ke arah versus. Net Shot: pukulan ini dilakukan dekat dengan net sebagai akibatnya diklaim sebagai net shot.

ISTILAH OFFICIAL Wasit Turnamen/ Tournament Referee: artinya seorang yang ditugaskan buat menentukan peraturan pada pertandingan. Bila pemain tidak sepakat dengan keputusan umpire, mereka bisa memanggil wasit turnamen. Match Umpire: tugas umpire buat memimpin serta memastikan pertandingan berjalan dengan lancar. Service Judge: duduk berseberangan menggunakan umpire dan tugasnya buat melihat servis yg dilakukan oleh pemain. Service judge berhak memutus service fault Bila pemain melakukan kesalahan ketika servis. Hakim Garis/ Line Judge: duduk di sekeliling lapangan dan bertugas buat melihat apakah bola jatuh pada luar atau di dalam lapangan.

11. Jumlah set dalam permainan bulu tangkis adalah.... a. 1 dan 3 set; b. 2 dan 3 set; c. 3 dan 4 set; d. 4 dan 6 set; e. 3 dan 5 set.


Dalam permainan bulu tangkis, setiap pertandingan terdiri dari (B) 2 dan 3 set. Set adalah jumlah babak dalam satu pertandingan bulu tangkis.

Pembahasan

Permainan bulu tangkis adalah salah satu olahraga bola besar yang dapat dimainkan secara tunggal atau beregu. Tujuan dari permainan bulu tangkis adalah untuk mematikan bola di area permainan lawan dan memenangkan pertandingan. Permainan bulu tangkis dimainkan di lapangan tertutup yang terhindar dari hembusan angin.

Dalam permainan bulu tangkis, setiap pertandingan dimainkan dalam 2 atau 3 babak. Jika pemain atau regu berhasil memenangkan 2 set awal, maka dinyatakan sebagai pemenang dan pertandingan berhasil. Namun jika kedua pemain atau regu imbang di 2 set awal, maka pertandingan dilanjutkan pada set ke 3 sebagai set penentuan.

Dalam permainan bulu tangkis, terdapat beberapa teknik dasar yang harus dikuasai oleh para pemain, yakni.

Teknik memegang raket Teknik posisi badan Teknik langkah kaki Teknik pukulan Teknik servis

12. Apa Perbedaan Set Shot Dan Jump Shot?


Jump shot adalah salah satu variasi teknik tembakan dalam permainan bola basket. Gerak jump shot diiringi lompatan saat melakukan tembakan. Tujuan dari gerakan ini adalah untuk menghindari usaha blocklawan terhadap gerakan shot yang Anda lakukan. 

Set-shoot adalah teknik shooting yang dilakukan tanpa melakukan lompatan, sedangkan jump-shoot adalah teknik shooting yang dilakukan dengan melompat.
