Cara menangguhkan update otomatis di Windows 10

Pertama-tama, klik menu "Start" kemudian masuk ke menu pengaturan (settings) dengan menekanklik ikon yang menyerupai "gir".

Atau bisa juga dengan mengetikkan kata kunci (keyword) "Settings" di kolom pencarian yang berada di sebelah tombol Start. Kemudian tekan tombol "Enter".

Nantinya, Anda akan disodori dengan berbagai menu pengaturan komputer. Di sini, pilih menu "Update & Security"Pada halaman selanjutnya, pilih menu "Windows Update" di deretan menu yang terpampang di sebelah kiri.

Lalu, pilih menu "Advanced options" di tampilan sebelah kanan.

Nah, di halaman selanjutnya, Anda bisa mematikan beberapa opsi yang tersedia, seperti "Receive updates for other..." dan "Download updates over...", dengan cara meng-klik kedua tombol tersebut menjadi "Off".

Kemudian, Anda juga perlu menentukan sampai kapan automatic updates akan ditangguhkan lewat menu drop-down "Select date" yang ditandai di gambar di bawah.

Device Manager dapat diakses melalui beberapa cara antara lain :

cara pertama : klik kanan pada My Computer, pilih properties, setelah jendela System terbuka pilih Device Manager dari list menu yang ada di sebelah kiri

cara kedua : tekan tombol windows + x, kemudian pilih Device Manager.


Untuk menginstal driver hardware secara manual pada windows, buka Device Manager terlebih dahulu,

setelah jendela Device Manager terbuka klik kanan pada My Computer kemudian pilih "Scan for hardware changes"lalu klik dua kali pada list yang bertuliskan others devices di windows 7 atau others di windows 10, di sini akan muncul daftar hardware yang belum memiliki driver.klik kanan pada nama hardware yang akan diinstal drivernya, lalu pilih update driverpilih search online jika anda tidak memiliki driver untuk hardware tersebut, atau pilih browse jika sudah memiliki driver tersebut

Yang perlu disiapkan untuk menggunakan driverpack solution untuk menginstal driver hardware adalah koneksi internet yang baik. Karena driverpack solution akan mencari driver yang sesuai melalui internet
