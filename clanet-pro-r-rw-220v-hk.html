2. apa kepanjangan dari.... a. CD-R b.ÇR-RW c. DVD-R d. DVD-RW


Kepanjangan dari CD-R, CD-RW, DVD-R, dan DV-RW yaitu:

CD-R yaitu Compact Disk - Read Only MemoryCD-RW yaitu Compact Disk - Read WriteDVD-R yaitu Digital Versatile Disk - Read Only MemoryDVD-RW yaitu Digital Versatile Disk - Read Write

Pembahasan

CD dan DVD merupakan hardware yang berungsi sebagai alat penyimpan data. CD dan DVD merupaka alat penyimpanan yang berupa piringan tipis. CD dan DVD terbagi menjadi dua yaitu:

CD-R (Compact Disk - Read Only Memory), berfungsi hanya untuk membaca sebuah CD saja. CD-R juga digunakan untuk melakukan penginstalasian sebuah OS (Operating System), Game, dan software lainnya.CD-RW (Compact Disk - Read Write), yaitu drive yang memiliki kemampuan membaca kepingan CD serta menulis di kepingan CD Blank. DVD-R (Digital Versatile Disk - Read Only Memory), yaitu drive yang memiliki fungsi untuk membaca data atau program pada DVD. DVD-RW (Digital Versatile Disk - Read Write), yaitu drive yang digunakan dalam tujuan backup, kumpulan berkas, atau home DVD video recorder. Keuntungan DVD-RW adalah data yang telah dimasukkan dapat dihapus lagi.

9. apa perbedaan cd r dan cd rw


Pertama yaitu CD-R. CD-R adalah singkatan dari Compact Disk Recordable adalah piringan pipih berbahan polikarbonat berdiameter 120 mm. Untuk mengisi CD-R dengan data diperlukan sebuah perangkat drive khusus. Proses perekamnya juga dilakukan dengan perangkat lunak khusus untuk mem burning data ke dalam CD-R. Data yang ditulis didalamnya bersifat WORM ( Write Once Read Many ) yang menunjukan bahwa data hanya dapat diisi sekali dan selanjutnya hanya bisa dilihat / dibaca.
Dapat diartikan jika kita telah burning data ke dalam CD-R, maka data tersebut sudah tidak bisa kita hapus lagi. Bisa dikatakan CD-R hanya dapat digunakan untuk satu kali pemakaian karena kita tidak bisa mengahapus ini data yang ada di dalamnya.

Kedua yaitu CD-RW yang merupakan pengembangan dari CD-R, sebenarnya dari bentuk sama dengan CD-R hanya sedikit berbeda dalam kegunaan dalam hal penyimpanan data. Perbedaanya adalah CD-RW ( Compact Disk Recordable and reWriteable ) berbahan CD-RW menggunakan logam perpaduan antara perak, indium, antimon, dan telurrium untuk lapisan perekaman dan dapat diubah datanya berkali - kali, dapat di tambah, diganti atau dihapus sewaktu - waktu.
Dapat diartikan jika kita telah memburning data kedalamnya, maka data tersebut masih bisa di hapus. Berbeda dengan CD-R, CD-RW bisa digunakan untuk menyimpan data berulang-ulang. Jadi seandainya  data yang ada pada CD-RW sudah tidak digunakan lagi, kita dapat menghapus dan menggantinya dengan yang baru.CD R tidak dapat mengopi file / video / gambar ke perangkat komputer atau laptop anda , dan harus melalui proses pembakaran dulu jika ingin menyalin video atau gambar tersebut pakai nero smart project, dan data yg telah disalin tidak dapat dihapus lagi, pemakaian hanya sekali saja.

CD RW dapat menyalin file / video / gambar ke perangkat laptop atau konputer anda tanpa melalui proses pembakaran /burning, dapat dipakai berulang ulang karena file dapat disalin ,dihapus, atau ditambahkan.

15. Berbeda dengan CD-R, CD-RW memiliki karakteristik seperti berikut ini,yaitu dapat


Berbeda dengan CD-R, CD-RW memiliki karakteristik seperti berikut ini,yaitu dapat ditulis kembali. CD atau Compact Disk adalah salah satu media penyimpanan pada komputer. Penggunaan media CD R maupun CD RW lebih aman dari virus. Kapasitas dari CD dapat menampung file hingga 800 mb.

Pengertian dari CD R adalah Compact Disk Recordable, CD R dapat kita isi dengan data sebanyak satu kali saja. Sedangkan pada CD RW, CD RW merupakan singkatan dari Compact Disk Recordable dan Rewritetable. Kita dapat melakukan penulisan data secara berulang-ulang. Dikarenakan fiturnya yang berbeda, maka harga dari CD R dan CD RW juga berbeda, harga CD RW biasanya hampir dua kali lipat dari CD R.

Demikian pembahasan mengenai media penyimpanan pada komputer yaitu CD R dan CD RW, terdapat media penyimpanan lainnya yaitu DVD, DVD R dengan DVD RW, untuk perbedaan dari keduanya sama dengan perbedaan pada CD R dan CD RW. Untuk sekarang sudah sangat jarang menyimpan file pada CD R atau CD RW karena kapasitas dari flashdisk atau micro sd juga sudah sangat besar.

Pada CD ROM terdapat tulisan 56X artinya kemampuan memberikan kecepatan transfer data sebesar 56 x150 Kbps. Tipe CD RW juga biasanya  dibedakan berdasarkan kemapuan membakar dan membaca. CD RW tipe 12x8x32 artinya memiliki kemampuan membakar pada CD R seccepat 12x, membakar pada CD RW secepat 8x, dan membaca CD R/CD RW/dengan kecepatan maksimal 32x.
2. CD-R adalah singkatan dari Compact Disk Recordable adalah piringan pipih berbahan polikarbonat berdiameter 120 mm. Untuk mengisi CD-R dengan data diperlukan sebuah perangkat drive khusus. Proses perekamnya juga dilakukan dengan perangkat lunak khusus untuk mem burning data ke dalam CD-R. Data yang ditulis didalamnya bersifat WORM ( Write Once Read Many ) yang menunjukan bahwa data hanya dapat diisi sekali dan selanjutnya hanya bisa dilihat / dibaca.
3. CD-RW merupakan pengembangan dari CD-R, sebenarnya dari bentuk sama dengan CD-R hanya sedikit berbeda dalam kegunaan dalam hal penyimpanan data. Perbedaanya adalah CD-RW ( Compact Disk Recordable and reWriteable ) berbahan CD-RW menggunakan logam perpaduan antara perak, indium, antimon, dan telurrium untuk lapisan perekaman dan dapat diubah datanya berkali - kali, dapat di tambah, diganti atau dihapus sewaktu - waktu.
