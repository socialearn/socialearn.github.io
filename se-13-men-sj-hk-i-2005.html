1. Tuliskan Hk. I Termodinamika..?


Apabila sistem gas menyerap kalor dari lingkungan sebesar Q1, maka oleh sistem mungkin akan diubah menjadi:
usaha luar (W) dan perubahan energi dalam ( Δ U),
energi dalam saja (U), dan
usaha luar saja (W).
Secara sistematis, peristiwa di atas dapat dinyatakan sebagai:

Q = W + U
Persamaan ini dikenal sebagai persamaan untuk hukum I Termodinamika. Bunyi hukum I Termodinamika adalah “Energi tidak dapat diciptakan ataupun dimusnahkan, melainkan hanya bisa diubah bentuknya saja.” Berdasarkan uraian tersebut terbukti bahwa kalor (Q) yg diserap sistem tidak hilang. Oleh sistem, kalor ini akan diubah menjadi usaha luar (W) dan atau penambahan energi dalam.

Penjelasan:Imbuhan

Imbuhan adalah merupakan suatu kata yang merupakan sambungan suatu kata yang terdiri dari:

Prefiks (awal) Infiks (tengah) Sufiks (akhir) Konfliks/ Simulfiks (awalan/akhir)

1. Prefiks (awal)

Prefiks/awal adalah suatu kata imbuhan yang terletak pada awal suatu kata.

Contoh

meN+ber+di+ter+peN+per+se+ke+

contoh kata

meN+masak = memasak (masak) meN+ngaji = mengaji (ngaji) meN+bantu = membantu (bantu) meN+baca = membaca (baca) dll

2. Infiks (tengah)

Infiks/tengah adalah suatu kata imbuhan yang terletak ditengah pada suatu kata.

Contoh

+el++em++er++e++in+

contoh kata

m+el+aju = melaju (maju) c+er+it+er+a = ceritera (cerita) g+em+etar = gemetar (getar) dll

3. Sufiks (akhir)

Sufiks/akhir adalah suatu kata imbuhan yang terletak diakhir pada suatu kata.

Contoh

+kan+an+i+nya

contoh kata

buat+an = buatan (buat) cinta+i = cintai (cinta) rasa+i = rasai (rasa)awal+an = awalan (awal) akhir+an = akhiran (akhir) dll

4. Konfliks/ Simulfiks (awalan/akhir)

Konfliks/Simulfiks adalah suatu kata imbuhan yang terletak diawal/diakhir pada suatu kata.

Contoh

ke+anpe+anme+kanpeN+anber+anse+nya

contoh kata

me+maaf+kan = memaafkan (maaf) ber+dasar+kan = berdasarkan (dasar) pel+ajar+an = pelajaran (belajar)

Pendahuluan :

Imbuhan adalah sebuah kata tambahan pada sebuah kalimat, Imbuhan biasanya terdapat pada :

Awal KalimatTengah KalimatAkhir KalimatAwalan dan Akhir

• Awal Kalimat

Imbuhan biasanya terletak pada Awal Kalimat atau dapat disebut sebagai Prefiks.

Contoh Imbuhan Awal Kalimat :

Men...Me...Ter...Bi...Ke...Di...dst..

Contoh kata yang sudah diberi Imbuhan :

Me + masak = MemasakDi + kebun = DikebunKe + pada = Kepadadst..

• Tengah Kalimat

Imbuhan biasanya terletak pada Tengah Kalimat atau dapat disebut sebagai Infiks.

Contoh Imbuhan Tengah Kalimat :

nya...atur...ambil...tan...dst..

Contoh kata yang sudah diberi Imbuhan :

Me + nya + pa = MenyapaPer + atur + an = PeraturanMeng + ambil + kan = MengambilkanBer + tan + nya = Bertanyadst..

• Akhir Kalimat

Imbuhan biasanya terletak pada Akhir Kalimat atau dapat disebut sebagai Sufiks.

Contoh Imbuhan Akhir Kalimat :

an...kan...ngi...nya...dst..

Contoh kata yang sudah diberi Imbuhan :

Ber + papas + an = BerpapasanMem + buat + kan = MembuatkanMe + lindu + ngi = Melindungidst..

• Awalan dan Akhir

Imbuhan biasanya terletak pada Awal dan Akhir atau dapat disebut sebagai Konfliks dan Simulfiks

Contoh Imbuhan Awal dan Akhir Kalimat :

Me + pa...Me + panen...Di + tanya...Se + bagai...dst..

Contoh kata yang sudah diberi Imbuhan :

Me + nga + pa = MengapaMe + panen = MemanenDi + tanya = DitanyaSe + bagai = Sebagaidst..

Pertanyaan :

1. meN- + takjub + kan = . . . .

2. meN- + pasar + kan = . . . .

3. meN- + karantina + kan = . . . .

4. meN- + tular + kan = . . . .

5. meN- + sanggup + i = . . . .

6. meN- + saksi + kan = . . . .

7. meN- + pangkas = . . . .

8. meN- + tiru + kan = . . . .

9. meN- + prakarsa + i = . . . .

10. meN- + seru + kan = . . . .

Pembahasan :

1. meN- + takjub + kan

￫ Menakjubkan

Kata Men-ak ( Awal Kalimat ) + kata takjub, kata "tak" dihapus ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

2. meN- + pasar + kan

￫ Memasarkan

Kata Men-ma ( Awal Kalimat ) + kata pasar, kata "p" dihapus ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

3. meN- + karantina + kan

￫ Mengkarantinakan

Kata Men-g ( Awal Kalimat ) + kata karantina ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

4. meN- + tular + kan

￫ Menularkan

Kata Men ( Awal Kalimat ) + kata tukar, huruf "t" digantikan dengan huruf "n" ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

5. meN- + sanggup + i

￫ Menyanggupi

Kata Men-nya ( Awal Kalimat ) + kata sanggup, kata "s" dihapus ( Tengah Kalimat ) + kata i ( Akhir Kalimat )

6. meN- + saksi + kan

￫ Menyaksikan

Kata Men-ya ( Awal Kalimat ) + kata saksi, kata "sa" dihapus ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

7. meN- + pangkas

￫ Memangkas

Kata Me ( Awal Kalimat ) + kata pangkas, kata "m" diubah menjadi kata "p" ( Akhir Kalimat )

8. meN- + tiru + kan

￫ Menirukan

Kata Me, kata "N" dihapus ( Awal Kalimat ) + kata tiru diubah menjadi kata "niru" ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

9. meN- + prakarsa + i

￫ Memperkarsai

Kata MeN, kata "N" diubah menjadi kata "m" ( Awal Kalimat ) + kata prakarsa ( Tengah Kalimat ) + kata i ( Akhir Kalimat )

10. meN- + seru + kan

￫ Menyerukan

Kata Men-y ( Awal Kalimat ) + kata seru, kata "s" dihapus ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

17. 13. Tahun 2005 menjadi tahun yang bersejarah bagi WordPress, dimana​


Penjelasan:

WordPress merupakan CMS (Content Management System) berbasis open source, yang sering digunakan sebagai engine blog, dibangun bersamaan dengan MySQL dan PHP, yang layanannya sendiri dapat Anda temukan melalui penyedia hosting murah. Jika Anda salah satu pengguna WordPress, tidak ada salahnya mengetahui tentang sejarah dan perkembangan WordPress berikut ini.

Sejarah WordPress

WordPress menjadi CMS paling populer saat ini, digunakan oleh lebih dari 12% dari 1 juta website yang tersebar di seluruh dunia. Hal ini dikarenakan WordPress memiliki banyak fitur, seperti template serta arsitektur plugin yang beragam. Pertama kali, WordPress dikembangkan oleh Matt Mullenweg yang saat ini masuk ke dalam jajaran 30 Under 30 inc.com, 25 Most Influential People on the Web Business Week, dan Top 50 People on the Web PC World.

Pada tahun 2002, Matt yang biasa menulis blog di b2 (bbpress.org) tidak beroperasi lagi, kemudian memiliki ide untuk mengembangkan blogware sendiri, namun tetap mengikuti standar website dengan struktur baik dan nuansa bersih. Keinginan tersebut semakin kuat saat tahun 2003 dimana Matt mengajak salah seorang rekannya, Mike Little untuk mulai mengembangkan WordPress. Hal ini pada akhirnya membuat pengembang b2, Michel Valdrighi, bergabung bersama Matt dan Mike Little.

Perkembangan WordPress

Pada tanggal 27 Mei 2003, versi pertama WordPress dirilis dan dilanjutkan dengan WordPress wiki atau lebih dikenal dengan codex.wordpress yang publish pada Desember 2003. Saat itu, WordPress mengalami popularitas yang terus meningkat, membuat CMS ini mendapatkan update besar-besaran pada Januari 2004, seperti menambahkan sejumlah fitur seperti edit link, multiple category, dan interface admin yang lebih baik dari versi sebelumnya. Nah, jika Anda juga sebagai salah satu pengguna WordPress, ada baiknya juga memilih layanan hosting murah yang bisa disesuaikan dengan budget maupun kebutuhan Anda.

Update pun dilanjutkan pada bulan Mei 2004, dimana saat perilisan versi 1.2, WordPress turut memperkenalkan fitur baru, seperti arsitektur plugin versi baru, sistem moderasi komentar, serta post review. Sementara tahun 2005 menjadi tahun yang bersejarah bagi WordPress, dimana versi 1.5 yang rilis pada tahun ini telah diunduh sebanyak 900 ribu kali, serta memulai service layanannya melalui wordpress.com.

Fitur lengkap dan kemudahan akses menjadi alasan mengapa CMS ini begitu populer di dunia. Jika Anda ingin mendapatkan kebutuhan hosting yang lengkap untuk membangun website, saatnya hubungi layanan hosting murah yang ada di Indonesia. Namun, pastikan pula bahwa layanan hosting yang Anda pilih tak hanya menawarkan harga ekonomis, namun juga memberikan pelayanan profesional.

23. Banyaknya bilangan bulat yang habis dibagi 13 antar 1000 dan 2005 adalah


Jawaban:

Penjelasan:

Kelipatan 13 antara 1000 dan 2005 :

1001, 1014, 1027, ..., 2002

a = 1001

b = 1014 - 1001 = 13

Un = 1001 + (n - 1)(13)

Un = 1001 + 13n - 13

Un = 988 + 13n

2002 = 988 + 13n

2002 - 988 = 13n

1014 = 13n

n = 1014/13 = 78

Banyaknya bilangan bulat yang habis dibagi 13 antar 1000 dan 2005 adalah 78 bilangan.
