23. pedagang bakpao menjual dua jenis bakpao.bakpao rasa ayam dijual dengan harga rp1.700,00 per buah dengan keuntungan rp200,00.bakpao rasa kacang dijual dengan harga rp1.100 per buah dengan keuntungan rp100,00.modal yang disediakan pedagang tersebut rp300.000,00.gerobak pedagang tersebut hanya mampu memuat 250 bakpao . pedagang tersebut akan mendapat keuntungan maksimum jiika memuat sebanyak .... bakpao rasa ayam dan ... bakpao rasa kacang.


Misalkan:
banyaknya bakpao ayam = x
banyaknya bakpao kacang = y.
x ≥ 0, y ≥ 0
Bakpao rasa ayam dijual dengan harga rp1.700,00 per buah dengan keuntungan rp200,00
(harga beli = 1.700 – 200 = 1.500).
Bakpao rasa kacang dijual dengan harga rp1.100 per buah dengan keuntungan rp100,00
(harga beli = 1.100 – 100 = 1.000).
Modal yang disediakan pedagang tersebut rp300.000,00.
→ 1.500x + 1.000y ≤ 300.000 ↔ 3x + 2y ≤ 600
Gerobak pedagang tersebut hanya mampu memuat 250 bakpao.
→ x + y ≤ 250
Bakpao rasa ayam dijual dengan keuntungan rp200,00. Bakpao rasa kacang dijual dengan keuntungan rp100,00.
→ f(x,y) = 200x + 100y
Akan ditentukan banyaknya jenis bakpao yang dijual agar didapat hasil keuntungan yang maksimum.
Gambarkan sistem pertidaksamaan berikut ke dalam bidang koordinat:
x ≥ 0
y ≥ 0
3x + 2y ≤ 600
x + y ≤ 250
Tentukan daerah solusi, sehingga diperoleh titik-titik uji solusi berikut.
(0,250) dari perpotongan x + y = 250 dengan sumbu y
(100,150) dari perpotongan 3x + 2y = 600 dan x + y = 250
(200,0) dari perpotongan 3x + 2y = 600 dengan sumbu x

Untuk (0,250) → f(0,250) = 200(0) + 100(250) = 0 + 25.000 = 25.000
Untuk (100,150) → f(100,150) = 200(100) + 100(150) = 20.000 + 15.000 = 35.000
Untuk (200,0) → f(200,0) = 200(200) + 100(0) = 40.000 + 0 = 40.000

Jadi, keuntungan maksimum sebesar 40.000 akan diperoleh pedagang jika membawa dan terjual 200 bakpao ayam.

Diketahui :

Harga Jual Bakpao Ayam   :Rp. 2.500,- dengan keuntungan Rp. 500,-

Harga Jual Bakpao Kacang:Rp. 2.000,- dengan keuntungan Rp. 300,-

Harga Beli : Rp. 2.000 - Rp. 300 = Rp. 1.700

→ x + y ≤ 250

→ f(x,y) = 500x + 300y

x ≥ 0

y ≥ 0
